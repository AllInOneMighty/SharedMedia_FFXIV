# SharedMedia: FFXIV

Installing this addon will allow you to use FFXIV Media (fonts, sounds, ...) in
any addon that utilizes the SharedMedia library.

SharedMedia is used in an estimated **~1,250+ addons** and downloaded over **100
million times**.

By installing SharedMedia_FFXIV, you add media to the list of media provided by
SharedMedia to other addons. If your other addons do not use SharedMedia, this
addon (SharedMedia_FFXIV) doesn't do anything.

Some popular addons that use SharedMedia:

- Recount / Skada Damage Meter / Details! Damage Meter
- Prat
- WeakAuras
- Omen Threat Meter
- Quartz
- …
